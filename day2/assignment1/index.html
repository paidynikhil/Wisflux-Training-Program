CSS

CSS (Cascading Style Sheets) is a style sheet language used for describing the presentation of a document written in HTML.

  • Basics:
    • CSS selectors
    • Colors/fonts/icons/links/lists
    • Backgrounds
    • Borders/box model
    • Margins/paddings
  • Intermediate:
    • Gradients
    • Shadows
    • transitions
    • Masking/pagination
  • Advance:
    • responsive
    • Animations
    • flex/grid
